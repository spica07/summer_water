# 전국 물놀이장 지도

전국 물놀이장(분수·물놀이 시설)을 지도에서 찾아보는 정적 웹앱입니다. `index.html` 한 장으로 동작하며 빌드 과정이 없습니다.

## 데이터 출처

- 각 지자체 공식 발표 및 보도자료 기준으로 개별 수집(시설별 출처는 `tools/facilities_extra.json`의 `source`/`sourceUrl`에 기록)
- 좌표는 주소 기반 카카오 로컬 API 지오코딩으로 보완
- 운영 기간·시간은 조사 시점 기준이며 방문 전 재확인이 필요합니다.

## 기술 스택

- Leaflet + OpenStreetMap 타일
- 빌드 없는 정적 HTML/CSS/JS, GitHub Pages(main 브랜치) 배포

## 안내 문구

본 사이트는 정보 공유 목적의 비영리 제공입니다. 무단 복제·재배포를 금합니다. 각 지자체 공식 발표·보도를 바탕으로 제작했으며,
표시되는 기관명·상표 등의 권리는 해당 권리자에게 있습니다. 위치 정보는 근사치일 수 있으니 참고용으로만 이용하세요.
